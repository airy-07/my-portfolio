## 📖 Deskripsi Proyek

Ini adalah repositori untuk **Website Portofolio Personal** saya. Website ini dirancang dengan gaya **Glassmorphism** modern yang menampilkan hobi saya di dunia pemrograman web, keahlian teknologi, serta proyek-proyek Next.js/Firebase yang sedang saya bangun secara interaktif.

🔗 **Link Demo Live:** [https://airy-07.github.io/my-portfolio/](https://airy-07.github.io/my-portfolio/)

---

## 🎨 Fitur Utama

- 🌓 **Mode Gelap/Terang (Dark/Light Mode)**: Beralih tema secara mulus dengan tombol toggle interaktif yang menyimpan preferensi tema pengguna di `localStorage`.
- 📱 **Desain Responsif**: Antarmuka modern yang secara dinamis beradaptasi sempurna pada layar perangkat mobile, tablet, maupun desktop.
- ✍️ **Efek Mesin Ketik (Typewriter Effect)**: Animasi mengetik dinamis pada bagian Hero untuk menyambut pengunjung.
- ⚙️ **Filter Proyek**: Menyaring kategori proyek latihan (Web App, Mobile App, dll.) secara interaktif tanpa memuat ulang halaman.
- 🌀 **Scroll Reveal Animations**: Efek kemunculan elemen dan bar keahlian yang teranimasi halus saat discroll (menggunakan *Intersection Observer API*).

---

## 🛠️ Teknologi yang Digunakan

- **HTML5**: Struktur web semantik yang teroptimasi SEO.
- **Vanilla CSS3**: Desain kustom menggunakan CSS Variables, Glassmorphism backdrop filter, flexbox/grid layout, dan animasi keyframe.
- **Vanilla JavaScript (ES6+)**: Logika interaktif penuh tanpa ketergantungan library eksternal (zero dependencies).
- **FontAwesome Icons**: Penyedia ikon media sosial dan antarmuka yang modern.
- **Google Fonts**: Menggunakan kombinasi font *Outfit* (Heading) dan *Inter* (Body).

## 💻 Cara Menjalankan Secara Lokal

Jika Anda ingin mencoba menjalankan atau memodifikasi website ini di komputer Anda sendiri, silakan ikuti langkah berikut:

1. **Clone Repositori:**
   ```bash
   git clone https://github.com/airy-07/my-portfolio.git
   ```

2. **Masuk ke Direktori Proyek:**
   ```bash
   cd my-portfolio
   ```

3. **Jalankan Server Lokal (Opsional):**
   Anda bisa membuka berkas `index.html` secara langsung di browser, atau menjalankannya melalui server lokal sederhana:
   *   Menggunakan Python:
       ```bash
       python -m http.server 8000
       ```
   *   Lalu buka: `http://localhost:8000` di browser Anda.
